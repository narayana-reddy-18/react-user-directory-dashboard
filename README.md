# React User Directory Dashboard

This is a small React application that fetches user data from a public API and displays it in a clean and responsive UI.

## Features
- Fetches user data from a public API
- Displays users in a card-based layout
- Search functionality to filter users
- Loading state while fetching data
- Error handling for API failures
- Responsive design

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## API Used
https://jsonplaceholder.typicode.com/users

## How to Run the Project

1. Clone the repository

git clone https://github.com/narayana-reddy-18/react-user-directory-dashboard.git

2. Navigate to the project folder

cd react-user-directory-dashboard

3. Install dependencies

npm install

4. Start the development server

npm start

The application will run at:
http://localhost:3000
